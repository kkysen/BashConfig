A simple package to add arrows and dashes


| Symbol | Shortcut |
|--------|----------|
| —     | `\--`, `\mdash`, or `\emdash`    |
| –     | `\ndash` or `\endash`    |
| <200b> (zero-width space)    | `\zwsp`  |
| •     | `\bul`   |
| ✓    | `\check` |
| §     | `\\sect` (Two slashes to avoid collision with LaTeX)    |
| →     | `\rarrow`    |
| ←     | `\larrow`    |
| ↑     | `\uarrow`    |
| ↓     | `\darrow`    |
| ⟷     | `\lrarrow`   |
